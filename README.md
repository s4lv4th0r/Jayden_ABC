# 🐴 ABC Caligrafia de Jayden

> Aplicacion educativa para que Jayden aprenda a escribir las letras del abecedario en español, con caligrafía escolar y un sistema de premios divertido.

🌐 **[Jugar aqui](https://s4lv4th0r.github.io/Jayden_ABC)**

---

## ✏️ Que puede hacer Jayden

- Repasar las **27 letras** del abecedario español (incluida la Ñ)
- Practicar en **3 modos**:
  - **A — Mayuscula**: repasa la letra en grande
  - **a — Minuscula**: repasa la letra pequeña
  - **Palabra**: escribe la palabra completa (Avion, Barco, Casa...)
- Pulsar **Validar** para que la app compruebe si lo ha hecho bien
- Ganar **monedas de oro** por cada letra correcta
- Ver a su **tia Ines** feliz o enfadada segun el resultado 😄😠

---

## 🏆 Sistema de premios

| Logro | Monedas |
|-------|---------|
| Letra mayuscula correcta | +3 monedas |
| Letra minuscula correcta | +3 monedas |
| Palabra completa correcta | +5 monedas |

## 🛠️ Tecnologia

- **React 18** via CDN (sin instalacion)
- **Playwrite ES** — fuente caligráfica escolar española (Google Fonts)
- **Canvas API** — para dibujar las letras con el dedo
- **CSS animations** — para las animaciones de premios
- **GitHub Pages** — hosting gratuito
